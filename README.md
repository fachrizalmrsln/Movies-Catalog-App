# Features
1. Showing data
    - Now Playing Movie
    - Upcoming Movie

2. Saving data to local database using SQLite
    - Saving favorite movie
    - Deleting favorite movie
   
3. Content Provide
    - Giving access to other module to get data from database
   
3. Other
   - Java
   - Design Pattern (MVP)
   - Glide
   - Swipe to refresh layout
